Popup menu hierarchy

Any of VARNA's interactive features can be accessed through a popup menu, invoked anywhere on the panel via a platform-dependent gesture (typically right mouse click, or Ctrl+left mouse click). Below is the current menu hierachy:

  • File
    • New... (Ctrl+N)
    • Open... (Ctrl+O)
    • Save... (Ctrl+S)
    • Export (Ctrl+X)
    • Print... (Ctrl+P)
  • Display
    • View
      • Zoom (Ctrl+Z)
      • Rotation... (Ctrl+R)
      • Border size (Ctrl+B)
    • RNA Style
      • Backbone color (Ctrl+K)
      • Base pairs
        • Base-pairs style... (Ctrl+Shift+P)
        • Show non-canonical BPs (Ctrl+W)
        • Hide tertiary base-pairs (Ctrl+E)
      • Non-standard bases
        • Color (Ctrl+Shift+J)
        • Custom colored (Ctrl+J)
        • Toggle draw bases (Alt+G)
      • 'Gaps' bases
        • Custom colored (Ctrl+D)
        • Color (Ctrl+Shift+D)
    • Color Map
      • Show color map (Ctrl+Shift+L)
      • Caption (Ctrl+Shift+C)
      • Style... (Ctrl+L)
      • Edit values... (Shift+L)
      • Load values... (Ctrl+Shift+K)
    • Misc
      • Background color (Ctrl+G)
      • Num. periodicity (Ctrl+M)
      • Title
        • Set title... (Ctrl+T)
        • Font (Ctrl+Shift+T)
        • Color (Ctrl+Alt+T)
  • Edit
    • Redraw
      • BP height increment (Ctrl+H)
      • Flat exterior loop (Ctrl+F)
      • Space between bases (Ctrl+Shift+S)
      • Reset (Ctrl+Shift+R)
      • Algorithm
        • Linear (Ctrl+1)
        • Circular (Ctrl+2)
        • Radiate (Ctrl+3)
        • NAView (Ctrl+4)
    • Colors
      • By base (Ctrl+U)
      • By BP (Shift+U)
      • By position (Alt+U)
    • Annotations
      • Edit from list...
      • Remove from list...
      • Auto helices (Ctrl+Q)
      • Auto interior loops (Alt+Shift+Q)
      • Auto terminal loops (Ctrl+Shift+Q)
      • New
        • Here (Ctrl+Shift+H)
        • Base
        • Loop
        • Helix
        • Region
        • Chem. Prob.
    • Selection
      • Current
      • Base
      • Base-pair
      • Bulge
      • Loop
      • Helix
      • Stem
      • All
  • About
